# Proyecto de Forecasting de Ventas

Estructura estándar para un proyecto de Machine Learning enfocado en forecasting de ventas.

## Estructura de Carpetas
- `data/raw/`: Datos originales sin procesar.
- `data/processed/`: Datos limpios y transformados.
- `notebooks/`: Jupyter Notebooks para exploración y modelado.
- `models/`: Modelos entrenados y scripts relacionados.
- `app/`: Aplicación Streamlit para visualización y predicción.
- `docs/`: Documentación adicional.

## Requisitos
Ver `requirements.txt` para dependencias.

## Uso rápido
1. Coloca los datos en `data/raw/`.
2. Usa los notebooks en `notebooks/` para análisis y modelado.
3. Guarda modelos en `models/`.
4. Ejecuta la app de Streamlit desde la carpeta `app/`.
