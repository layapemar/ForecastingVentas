# Estructura del Proyecto

- **data/raw/**: Datos originales sin procesar.
- **data/processed/**: Datos limpios y listos para modelar.
- **notebooks/**: Notebooks de Jupyter para análisis y modelado.
- **models/**: Modelos entrenados y scripts relacionados.
- **app/**: Aplicación Streamlit para visualización y predicción.
- **docs/**: Documentación y guías del proyecto.
